# A  Oil & Gas Downstream Operations Dashboard

## Project Overview
This Excel dashboard provides an interactive view of an **Oil & Gas downstream operations**, highlighting key performance metrics and visual insights. The dashboard is designed to help managers and analysts quickly understand product performance, depot contributions, and revenue trends.

## Key Features
- **KPI Cards**: Total Revenue, Total Volume, Average Margin, Top Product, and Top Depot — each with icons for quick interpretation.
- **Charts**:
  - Revenue by Product (Column Chart)
  - Revenue by Depot (Bar Chart)
  - Monthly Revenue Trend (Line Chart)
  - Revenue by Customer Type (Donut Chart)
- **Interactive Slicers**: Filter by Depot, Product, and Month to update KPIs and charts in real time.
- **Professional Layout**: Clean background, color-coded KPI cards, and icons for clarity.

## Tools & Skills
- **Microsoft Excel** (Pivot Tables, Pivot Charts, Slicers)
- **Data Visualization**
- **Dashboard Design**
- **Portfolio Project** showcasing analytical and visualization skills

## How to Use
1. Download the Excel file (`Oil& Gas Dashboard.xlsx`) from this repository.
2. Open the file in Excel.
3. Use the **slicers** to filter the data and observe the KPI and chart updates.
4. Explore trends, top-performing products, and depots using interactive visualizations.
